# My Creative Blog

A blog built with HTML, CSS, and JavaScript - learning to code while building something beautiful!

## Files

- **index.html** - The structure of your blog (HTML)
- **styles.css** - The design and styling (CSS)
- **script.js** - Interactive features (JavaScript)

## Getting Started

### View Your Blog Locally

1. In VS Code, right-click on `index.html`
2. Select "Open with Live Server" (install the extension if needed)
3. Your blog will open in your browser

## What You Can Customize

### Colors
In `styles.css`, find this section and change the colors:
```css
:root {
    --primary-color: #2c3e50;      /* Main dark color */
    --secondary-color: #3498db;    /* Accent blue */
    --accent-color: #e74c3c;       /* Accent red */
}
```

Use [coolors.co](https://coolors.co) to find colors you love!

### Fonts
Change the font in `styles.css`:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```

Try fonts from [Google Fonts](https://fonts.google.com)

### Content
Edit the text in `index.html` to make it your own!
